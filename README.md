# Nifty100 Financial Data Warehouse

## Overview

ETL pipeline built using Python and SQLite for loading and validating financial data from multiple Excel files.

## Features

- Excel Loader
- Data Normalization
- Data Validation
- SQLite Database
- Load Audit Report
- Validation Report

## Tech Stack

- SQLite
- SQL
- OpenPyXL
